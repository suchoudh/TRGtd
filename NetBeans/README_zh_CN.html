NetBeansTM IDE

NetBeans IDE 是一个集成开发环境 (Integrated Development Environment, IDE)，可用来编写、编译、测试和调试 JavaTM 平台及其他环境的软件应用程序。NetBeans IDE 包含一个功能完善的文本编辑器、可视设计工具、源代码管理支持、数据库集成工具以及许多其他功能。

可以采用以下某种方法来启动 IDE (Microsoft Windows)：

  • 双击桌面上的 NetBeans IDE 图标。
  • 选择“开始”>“所有程序”> "NetBeans" > "NetBeans IDE 6.5"。
  • 通过命令行启动 IDE：C:\> netbeans-install-directory\bin\netbeans.exe。

启动 IDE（Solaris OS 或 Linux）：

  • 在命令提示符下键入：/netbeans-install-directory/bin/netbeans

停止 IDE：

  • 从 IDE 中，选择“文件”>“退出”。

卸载 IDE (Microsoft Windows)

  • 使用“添加/删除程序”实用程序。请不要手动删除目录和文件。

卸载 IDE（Solaris OS 或 Linux）

使用 /netbeans-install-directory/uninstall.sh 中提供的卸载程序。请不要手动删除目录和文件。

查找 NetBeans IDE 文档

  • 在以下位置可以找到有关 NetBeans IDE 6.5 的发行信息：http://www.netbeans.org/community/releases/65/index_zh_CN.html。
  • 有关 IDE 文档的索引，请访问http://www.netbeans.org/kb/index_zh_CN.html。该文档还提供了指向其他文档的链接。
  • IDE 中提供了帮助文档。您可以选择主菜单中的“帮助”>“帮助内容”进行查看。此外，“帮助”菜单还列出了指向其他相关文档的链接。

NetBeans IDE 用户目录

第一次启动 IDE 时，会创建一个缺省的用户目录。它用于为个人开发服务器存储您的用户设置、选项和运行时数据。在 Microsoft Windows Vista 系统上，此目录通常位于 C:\Users\user-id\.netbeans\version。在其他 Microsoft Windows 系统上，此目录通常位于 C:\Documents and Settings\user-id\.netbeans\version。在 Solaris 和 Linux 上，此目录位于 /$HOME/netbeans_user。
