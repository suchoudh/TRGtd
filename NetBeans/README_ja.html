NetBeansTM IDE

NetBeans IDE は、JavaTM プラットフォームおよびその他の環境用のソフトウェアアプリケーションを作成、コンパイル、テスト、およびデバッグするための統合開発環境 (IDE) です。NetBeans IDE には、十分な機能を備えたテキストエディタ、ビジュアルデザインツール、ソースコード管理サポート、データベース統合ツールなどの多くの機能があります。

Microsoft Windows の場合、IDE は次のどの方法でも起動できます。

  • デスクトップ上の NetBeans IDE のアイコンをダブルクリックする。
  • 「スタート」>「すべてのプログラム」>「NetBeans」>「NetBeans IDE 6.5」を選択する。
  • IDE をコマンド行 (C:\> netbeans-install-directory\bin\netbeans.exe) から起動する。

Solaris OS または Linux の場合は、次の方法で IDE を起動できます。

  • コマンドプロンプトで「 /netbeans-install-directory/bin/netbeans」と入力する。

IDE を終了する方法は、次のとおりです。

  • IDE から「ファイル」>「終了」を選択する。

Microsoft Windows の場合、IDE をアンインストールする方法は次のとおりです。

  • 「アプリケーションの追加と削除」ユーティリティーを使用する。ディレクトリやファイルを手動で削除しないでください。

Solaris OS または Linux の場合、IDE をアンインストールする方法は次のとおりです。

/netbeans-install-directory/uninstall.sh にある付属のアンインストーラを使用する。ディレクトリやファイルを手動で削除しないでください。

NetBeans IDE に関するマニュアル等の入手場所

  • NetBeans IDE 6.5 のリリース情報は、http://www.netbeans.org/community/releases/65/index.html を参照してください。
  • IDE マニュアルの索引は、http://www.netbeans.org/kb/index_ja.html を参照してください。このマニュアルは、ほかのマニュアルへのリンクも含まれています。
  • IDE には、ヘルプが用意されています。メインメニューから「ヘルプ」>「ヘルプの目次」を選択してください。「ヘルプ」メニューにも、ほかの有用なドキュメントへのリンクが含まれています。

NetBeans IDE ユーザーディレクトリ

デフォルトのユーザーディレクトリは、IDE を最初に起動したときに作成されます。これはユーザーの構成、オプション、配備サーバーの実行時データを保存するために使用されます。Microsoft Windows Vista システムの場合、このディレクトリは通常 C:\Users\user-id\.netbeans\version にあります。Microsoft Windows システムの場合、このディレクトリは通常 C:\Documents and Settings\user-id\.netbeans\version にあります。Solaris および Linux の場合、このディレクトリは /$HOME/netbeans_user にあります。
