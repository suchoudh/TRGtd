GUI フォームの例の説明

このプロジェクトには IDE の GUI ビルダーでデザインされたビジュアルフォームが 3 つ含まれています。これらのフォームは IDE で開くことができます。また、IDE でこれらのフォームを作成する方法を示した付属のデモを表示できます。

GUI ビルダーでフォーム例を開く:

  1. IDE の「プロジェクト」ウィンドウで、「ソースパッケージ」ディレクトリ内のデフォルトのパッケージノードに移動します。

  2. 表示するサンプルフォームのノードをダブルクリックします。

    IDE が GUI ビルダーを起動し、選択したフォームを表示します。


コンタクトエディタ

この複合フォームは、データベース情報の表示と編集に通常利用される Windows のダイアログに似ています。この例では、整列、アンカー、サイズ変更、インデント、および実行時の自動サイズ変更動作など、GUI ビルダー機能の全般を説明しています。このフォームの構築方法を示した Flash のデモが含まれている詳しいチュートリアルを表示するには、GUI Building in NetBeans IDE 5.0 を参照してください。


アンテナ設定の変更

この例では、見た目に分かりやすいグループ形式で関連する情報を表示するために一般的に利用される、表ベースのフォームの一種を説明しています。GUI ビルダーの拡張整列機能を使うと、隣接しないコンポーネント同士 (ラベル、テキストフィールド、およびボタンなど) のグループ化とサイズ変更が、セルを手動でマージする必要なく一貫して行えます。このフォームの構築方法を示した Flash のデモを表示するには、GUI Form Example: Antenna Preferences Dialog を参照してください。


検索

このシンプルなフォームでは、基本的な検索機能を必要とする各種アプリケーションでよく使われる、単純なレイアウトを説明しています。GUI ビルダーの整列およびサイズ変更機能を使うと、このようなフォームをすばやく作成する一方で、配置を確実に予測できます。このフォームの構築方法を示した Flash のデモを表示するには、GUI Form Example: Find Dialog を参照してください。
