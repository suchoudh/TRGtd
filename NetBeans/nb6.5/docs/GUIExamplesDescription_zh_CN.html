GUI 窗体示例描述

此项目包含三个使用 IDE 的 GUI 生成器设计的可视窗体。您可以在 IDE 中打开这些窗体，并且可以查看说明如何在 IDE 中创建这些窗体的附带演示。

在 GUI 生成器中打开示例窗体：

  1. 在 IDE 的“项目”窗口中，导航至缺省包节点，该节点位于
    “源包”目录。

  2. 双击要查看的样例窗体的节点。

    IDE 将启动 GUI 生成器并显示所选的窗体。


联系人编辑器

此复杂窗体类似于通常用于显示和编缉数据库信息的 Windows 对话框。此示例说明了各种 GUI 生成器功能，其中包括对齐、锚点、调整大小、缩排以及运行时自动调整大小行为。要查看详细教程（包括说明如何生成此窗体的动画演示），请参见在 NetBeans IDE 5.0 中生成 GUI。


Antenna 首选项

此示例说明了一种基于表的窗体，它通常用于在可视相关组中显示相关信息。利用 GUI 生成器的高级对齐功能，可以将不相邻的组件（如标签、文本字段和按钮）进行分组并保证大小一致，而无需手动合并单元格。要查看说明如何生成此窗体的动画演示，请参见 GUI 窗体示例：“Antenna 首选项”对话框。


查找

此简单窗体说明了一种直观的布局，它通常用于各种需要基本搜索功能的应用程序。利用 GUI 生成器的对齐和调整大小功能，可以快速构造此类窗体，同时可确保按照可预测的方式本地化这些窗体。要查看说明如何生成此窗体的动画演示，请参见 GUI 窗体示例：“查找”对话框。
