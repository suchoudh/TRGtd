Descrições de exemplos de formulários de GUI

Este projeto contém três formulários visuais criados com o construtor de GUI do IDE. Estes formulários podem ser abertos no IDE e podem ser vistos acompanhados de demonstrações que mostram como tais formulários podem ser criados no IDE.

Para abrir um formulário de exemplo no construtor de GUI:

  1. Na janela Projetos do IDE, vá até o nó do pacote padrão no
    diretório Pacotes de código-fonte.

  2. Clique duas vezes no nó do formulário de amostra que deseja visualizar

    O IDE inicia o construtor de GUI e exibe o formulário escolhido.


Editor de contato

Este formulário complexo é semelhante às caixas de diálogo do Windows normalmente usadas para exibir e editar informações de bancos de dados. O exemplo ilustra todo a série de recursos do construtor de GUI, incluindo alinhamento, ancoramento, dimensionamento, indentação e comportamento de redimensionamento automático em tempo de execução. Para visualizar um tutorial detalhado incluindo demonstrações em flash que mostra como construir este formulário, consulte Construindo GUIs no NetBeans IDE 5.0.


Preferências de Antenna

Este exemplo ilustra um tipo de formulário baseado em tabela usado normalmente para exibir informações relacionadas em grupos visualmente lógicos. Os recursos avançados de alinhamento do construtor de GUI permite agrupar e dimensionar os componentes não adjacentes (como rótulos, campos de texto e botões) de forma coerente sem ter que mesclar as células manualmente. Para visualizar uma demonstração em flash que mostra como construir este formulário, consulte Exemplo de formulário de GUI: Caixa de diálogo Preferências de Antenna.


Localizar

Este formulário simples ilustra um layout comum usado freqüentemente em vários aplicativos que requerem a funcionalidade de pesquisa básica. Os recursos de alinhamento e dimensionamento do construtor de GUI permitem construir tais formulários rapidamente e garantir que eles estejam bem situados. Para visualizar uma demonstração em flash que mostra como construir este formulário, consulte Exemplo de formulário de GUI: Caixa de diálogo Localizar.
