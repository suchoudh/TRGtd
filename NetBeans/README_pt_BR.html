NetBeansTM IDE

O NetBeans IDE é um ambiente de desenvolvimento integrado (IDE) para escrever, compilar, testar e depurar aplicativos de software para a plataforma JavaTM e outros ambientes. O NetBeans IDE inclui um editor de texto completo, ferramentas de visual design, suporte ao gerenciamento de código-fonte, ferramentas de integração de banco de dados e muitos outros recursos.

Para iniciar o IDE (Microsoft Windows) use um dos métodos seguintes:

  • Clique duas vezes no ícone do NetBeans IDE na área de trabalho.
  • Escolha Iniciar > Todos os programas > NetBeans > NetBeans IDE 6.5.
  • Inicie o IDE na linha de comando C:\> netbeans-install-directory\bin\netbeans.exe.

Para iniciar o IDE (Solaris OS ou Linux):

  • Digite no prompt de comando: /netbeans-install-directory/bin/netbeans

Para parar o IDE:

  • No IDE, escolha Arquivo > Sair.

Para desinstalar o (Microsoft Windows)

  • Use o utilitário Adicionar/Remover programas. Não exclua os diretórios e arquivos manualmente.

Para desinstalar o IDE (Solaris OS ou Linux)

Use o desinstalador fornecido encontrado em /netbeans-install-directory/uninstall.sh. Não exclua os diretórios e arquivos manualmente.

Procurando documentação do NetBeans IDE

  • Informações sobre os lançamentos do NetBeans IDE 6.5 podem ser encontradas em http://www.netbeans.org/community/releases/65/index.html.
  • É possível encontrar o índice da documentação do IDE em http://www.netbeans.org/kb/index.html. Este documento também fornece links para futuras documentações.
  • A documentação de ajuda está disponível no IDE. No menu principal, escolha Ajuda > Conteúdo da ajuda. O menu Ajuda também fornece links para outras documentações úteis.

Diretório de usuário do NetBeans IDE

Um diretório de usuário padrão é criado na primeira vez que o IDE é iniciado. É usado para armazenar configurações, opções e dados em tempo de execução do usuário em relação aos servidores pessoais de desenvolvimento. Em sistemas Microsoft Windows Vista, o diretório está geralmente localizado em C:\Users\user-id\.netbeans\version. Em sistemas Microsoft Windows, este diretório encontra-se em C:\Documents and Settings\user-id\.netbeans\version. No Solaris e no Linux, este diretório encontra-se em /$HOME/netbeans_user
